# Introduction to Credit Risk and Applications in Python

## Overview

This repository contains teaching materials for the course *Introduction to Credit Risk and Applications in Python*. The content is intended to support learning and understanding of fundamental concepts in credit risk, along with practical implementation in Python.

The repository is continuously updated, and materials may evolve over time.
